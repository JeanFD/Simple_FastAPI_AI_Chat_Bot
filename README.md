# Simple FastAPI AI Chat Bot

Chat Bot simples com **FastAPI** e **Llama 3** (via Groq).

## Como rodar

### Backend
1. Instale as dependências: `pip install fastapi uvicorn groq python-dotenv`
2. Configure a `GROQ_API_KEY` no arquivo `.env`.
3. Rode: `uvicorn main:app --reload` (dentro da pasta `backend`).

### Frontend
Abra o `index.html` no navegador.

## Techs
FastAPI, Groq/Llama 3, HTML/JS.
